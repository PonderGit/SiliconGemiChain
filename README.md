To ensure the reliability of AI-generated responses (In Silico samples), this study employs a standardized code executed via Google Colab. The code automates the transition from raw survey data to validated datasets.Technical Steps:
1 Data Preprocessing:The code performs automated cleaning and mapping of CSV files. Categorical data from 5-7 point Likert scales are treated as interval data, grounded in Norman (2010), which confirms the robustness of parametric statistics for such measurements.
2 Adaptive Generation:Utilizing the DeepSeek-V3 API, the code generates synthetic counterparts for respondents. It injects adaptive perturbations into the prompts to prevent over-fitting and capture the inherent stochasticity of human subjectivity.
3 Statistical Validation:The code executes three modules to assess Algorithmic Fidelity:Distributional Alignment: Conducts Wilcoxon Tests to confirm no significant difference ($p > 0.05$) between human and synthetic means.Structural Alignment: Verifies the preservation of covariance structures using Spearman Correlation and Fisher-z transformations.Group Alignment: Implements Kruskal-Wallis tests with 95% Bootstrapped Confidence Intervals to ensure consistent inter-group dynamics.
